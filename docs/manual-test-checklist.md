# 수동 테스트 체크리스트 - 상세페이지 기본 정보 섹션

> 개발 서버 실행 후 브라우저에서 직접 확인하는 체크리스트입니다.

## 사전 준비

1. 개발 서버 실행
   ```bash
   pnpm dev
   ```
2. 브라우저에서 `http://localhost:3000` 접속

---

## 1단계: 홈페이지에서 상세페이지 접근 확인

### 체크사항

- [ ] 홈페이지(`/`)가 정상적으로 로드되는지 확인
- [ ] 관광지 목록이 표시되는지 확인
- [ ] 관광지 카드 중 하나를 클릭
- [ ] 상세페이지(`/places/[contentId]`)로 이동되는지 확인
- [ ] 페이지가 정상적으로 로드되는지 확인 (에러 화면이 아닌지)

**예상 결과**: 관광지 카드를 클릭하면 상세페이지로 이동하고, 기본 정보가 표시됩니다.

---

## 2단계: 기본 정보 표시 확인

### 2-1. 관광지명 및 뱃지

- [ ] 관광지명이 큰 제목으로 표시되는지 확인
- [ ] 관광 타입 뱃지가 표시되는지 확인 (예: "관광지", "문화시설", "음식점" 등)
  - 뱃지 색상: 파란색 계열 배경에 파란색 텍스트
- [ ] 카테고리 뱃지가 표시되는지 확인 (예: "자연", "산", "계곡" 등)
  - 뱃지 스타일: 회색 배경에 테두리 (타입 뱃지와 구분됨)
  - cat1, cat2, cat3가 있으면 모두 표시되는지 확인

**예상 결과**: 
- 관광지명이 페이지 상단에 크게 표시됨
- 타입 뱃지와 카테고리 뱃지가 관광지명 위에 표시됨
- 뱃지들이 가로로 나열되어 있음

---

### 2-2. 대표 이미지

- [ ] 대표 이미지가 표시되는지 확인
- [ ] 이미지 비율이 적절한지 확인 (가로로 긴 직사각형)
- [ ] 이미지가 깨지지 않고 정상적으로 보이는지 확인
- [ ] 이미지가 없는 경우 "이미지 없음" 메시지가 표시되는지 확인

**예상 결과**: 
- 이미지가 있으면 큰 이미지로 표시됨
- 이미지가 없으면 회색 배경에 "이미지 없음" 텍스트가 표시됨

---

### 2-3. 주소 정보 및 복사 기능

- [ ] 주소가 표시되는지 확인
  - 주소 아이콘(📍)이 있는지 확인
  - 주소 텍스트가 정상적으로 표시되는지 확인
- [ ] 주소 옆에 복사 버튼(📋 아이콘)이 있는지 확인
- [ ] 복사 버튼을 클릭
  - 버튼 아이콘이 체크 표시(✓)로 바뀌는지 확인
  - 화면 하단 또는 상단에 토스트 메시지가 나타나는지 확인
    - 메시지 내용: "주소가 클립보드에 복사되었습니다"
- [ ] 복사 후 다른 곳(메모장, 검색창 등)에 붙여넣기(Ctrl+V 또는 우클릭 → 붙여넣기)
  - 주소가 정상적으로 붙여넣어지는지 확인
- [ ] 2초 후 복사 버튼 아이콘이 다시 복사 아이콘으로 돌아오는지 확인

**예상 결과**: 
- 주소가 표시되고, 복사 버튼을 클릭하면 주소가 클립보드에 복사됨
- 복사 성공 시 토스트 메시지가 나타남
- 복사된 주소를 다른 곳에 붙여넣을 수 있음

---

### 2-4. 전화번호

- [ ] 전화번호가 표시되는지 확인
  - 전화 아이콘(📞)이 있는지 확인
  - 전화번호 텍스트가 정상적으로 표시되는지 확인
- [ ] 전화번호를 클릭
  - 모바일: 전화 앱이 열리는지 확인
  - 데스크톱: 전화 앱이 열리거나 전화 연결 시도가 되는지 확인
- [ ] 전화번호가 없는 경우 해당 섹션이 표시되지 않는지 확인

**예상 결과**: 
- 전화번호가 있으면 파란색 링크로 표시됨
- 클릭하면 전화 연결이 시도됨
- 전화번호가 없으면 해당 섹션이 아예 표시되지 않음

---

### 2-5. 홈페이지

- [ ] 홈페이지 링크가 표시되는지 확인
  - 지구본 아이콘(🌐)이 있는지 확인
  - 홈페이지 URL이 정상적으로 표시되는지 확인
  - 외부 링크 아이콘(↗)이 있는지 확인
- [ ] 홈페이지 링크를 클릭
  - 새 창(또는 새 탭)에서 홈페이지가 열리는지 확인
- [ ] 홈페이지가 없는 경우 해당 섹션이 표시되지 않는지 확인

**예상 결과**: 
- 홈페이지가 있으면 파란색 링크로 표시되고, 클릭하면 새 창에서 열림
- 홈페이지가 없으면 해당 섹션이 아예 표시되지 않음

---

### 2-6. 개요

- [ ] 개요 섹션이 별도의 카드로 표시되는지 확인
- [ ] 개요 제목이 "개요"로 표시되는지 확인
- [ ] 개요 내용이 긴 설명문으로 표시되는지 확인
- [ ] 줄바꿈이 정상적으로 표시되는지 확인
- [ ] 개요가 없는 경우 해당 섹션이 표시되지 않는지 확인

**예상 결과**: 
- 개요가 있으면 별도의 카드에 표시됨
- 긴 설명문이 여러 줄로 정상적으로 표시됨
- 개요가 없으면 해당 섹션이 아예 표시되지 않음

---

## 3단계: 다양한 관광지로 테스트

### 체크사항

다양한 관광지를 클릭하여 다음을 확인:

- [ ] 이미지가 있는 관광지
- [ ] 이미지가 없는 관광지
- [ ] 전화번호가 있는 관광지
- [ ] 전화번호가 없는 관광지
- [ ] 홈페이지가 있는 관광지
- [ ] 홈페이지가 없는 관광지
- [ ] 개요가 긴 관광지
- [ ] 개요가 없는 관광지
- [ ] 카테고리 정보(cat1, cat2, cat3)가 있는 관광지
- [ ] 카테고리 정보가 없는 관광지

**예상 결과**: 
- 정보가 있는 항목만 표시되고, 없는 항목은 숨김 처리됨
- 모든 경우에서 페이지가 정상적으로 로드됨

---

## 4단계: 반응형 디자인 확인

### 4-1. 데스크톱 화면 (1920px 이상)

- [ ] 모든 정보가 한 줄에 적절히 표시되는지 확인
- [ ] 이미지가 적절한 크기로 표시되는지 확인
- [ ] 뱃지들이 가로로 나열되는지 확인

### 4-2. 태블릿 화면 (768px ~ 1024px)

- [ ] 브라우저 창 크기를 조절하여 태블릿 크기로 변경
- [ ] 레이아웃이 적절히 조정되는지 확인
- [ ] 텍스트와 이미지가 읽기 쉽게 표시되는지 확인

### 4-3. 모바일 화면 (375px ~ 768px)

- [ ] 브라우저 개발자 도구에서 모바일 뷰로 전환 (F12 → 디바이스 툴바)
- [ ] 또는 브라우저 창을 모바일 크기로 축소
- [ ] 모든 정보가 세로로 적절히 배치되는지 확인
- [ ] 복사 버튼이 터치하기 쉬운 크기인지 확인
- [ ] 텍스트가 작지 않고 읽기 쉬운지 확인

**예상 결과**: 
- 모든 화면 크기에서 레이아웃이 적절히 조정됨
- 모바일에서도 모든 기능이 정상적으로 동작함

---

## 5단계: 에러 처리 확인

### 체크사항

- [ ] 존재하지 않는 관광지 ID로 접근 시도
  - URL에 잘못된 contentId 입력 (예: `/places/999999999`)
  - 에러 메시지가 표시되는지 확인
  - "관광지를 찾을 수 없습니다" 또는 유사한 메시지가 표시되는지 확인
  - "홈으로 돌아가기" 버튼이 있는지 확인

**예상 결과**: 
- 존재하지 않는 관광지에 접근하면 친화적인 에러 메시지가 표시됨
- 홈으로 돌아갈 수 있는 버튼이 제공됨

---

## 6단계: 뒤로가기 버튼 확인

### 체크사항

- [ ] 상세페이지 상단에 "목록으로 돌아가기" 버튼이 있는지 확인
- [ ] 버튼을 클릭
  - 홈페이지로 돌아가는지 확인
- [ ] 브라우저의 뒤로가기 버튼도 정상 작동하는지 확인

**예상 결과**: 
- 뒤로가기 버튼을 클릭하면 홈페이지로 이동함
- 브라우저의 뒤로가기 버튼도 정상 작동함

---

## 빠른 확인 체크리스트 (요약)

다음 항목들을 빠르게 확인하면 됩니다:

### 필수 확인 사항 (5분)

- [ ] **상세페이지 접근**: 홈에서 관광지 클릭 → 상세페이지 이동 확인
- [ ] **기본 정보 표시**: 관광지명, 이미지, 주소, 전화번호, 홈페이지, 개요가 모두 표시되는지 확인
- [ ] **주소 복사 기능**: 주소 옆 복사 버튼 클릭 → 토스트 메시지 확인 → 붙여넣기로 복사 확인
- [ ] **뱃지 표시**: 타입 뱃지와 카테고리 뱃지가 표시되는지 확인
- [ ] **정보 없는 항목 숨김**: 전화번호나 홈페이지가 없는 관광지에서 해당 섹션이 숨겨지는지 확인

### 추가 확인 사항 (선택, 10분)

- [ ] **반응형 디자인**: 모바일 크기로 창을 줄여서 레이아웃이 적절히 조정되는지 확인
- [ ] **다양한 관광지**: 이미지 없음, 정보 일부 없음 등 다양한 케이스 확인
- [ ] **에러 처리**: 존재하지 않는 관광지 ID로 접근 시 에러 메시지 확인

## 발견한 문제가 있다면?

다음 정보를 기록해주세요:

1. **어떤 화면에서 문제가 발생했나요?** (홈, 상세페이지 등)
2. **무엇을 했을 때 문제가 발생했나요?** (클릭, 스크롤 등)
3. **예상했던 동작은 무엇이었나요?**
4. **실제로 어떤 일이 발생했나요?** (에러 메시지, 화면 깨짐 등)
5. **브라우저와 화면 크기**: (예: Chrome, 1920x1080)
